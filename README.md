## Agents / Skills

### [Call Analysis](./agents/call-analysis/)
Reads meeting transcripts and extracts decisions, open questions, and action items — structured and ready to share.

### [Telegram Notes](./agents/telegram-notes/)
Collects voice and text messages from a Telegram bot, transcribes audio with Whisper, and classifies everything into Ideas and Tasks.

### [Weekly Content](./agents/weekly-content/)
The meta-agent. Give it a topic → it generates a LinkedIn post and a YouTube presentation. What you're looking at right now was made with this.

---

## How to use these skills

1. Install Claude Code → see [`setup/getting-started.md`](./setup/getting-started.md)
2. Copy the `SKILL.md` file from any skill folder into `~/.claude/commands/` on your machine (rename it to `[skill-name].md`)
3. Restart Claude Code and run `/<skill-name>`
