I've been saying "I should start writing about AI" for months.

And today, you know, an AI Agent helped me stop saying it and actually do it. 😃

Two weeks ago I started seriously learning Claude Agents. And the first thing I built was for myself — a personal content agent.

It doesn't just generate content. It:
- Analyses my notes from learning materials
- Translates my personal notes from Telegram (even voice ones)
- Generates my weekly content: LinkedIn post, YouTube slides, and speaker notes for recording on camera

(For a long time I wanted to improve my presentation skills — now I have a script every week. No more excuses 😄)

I called my project folder "AI girl Anastasia." Honestly? It makes me feel powerful.

Here's how it works each week:

1. Open terminal in my project folder
2. Run Claude Code
3. Type /weekly-content

The skill has 3 sections:

**Persona** — tells Claude who I am, what my voice sounds like, and who reads my content. Every time I run it, Claude already knows my tone. Hopefully this is what will make it sound like me, not generic AI 😊

**"Before you start"** — before writing anything, it asks me 4 questions:
- What's the topic this week?
- What's the post number?
- Do I have a real story to share?
- Anything to avoid?

**Output** — 3 files saved automatically to the right folders:
→ posts/01-your-topic/linkedin.md
→ posts/01-your-topic/slides.html
→ posts/01-your-topic/speaker-notes.md

In the future I want to improve it — instead of asking me manually, it will just redirect to my notes folder for the week's topic.

This is week 1. Let's see how consistent I can be 😄
