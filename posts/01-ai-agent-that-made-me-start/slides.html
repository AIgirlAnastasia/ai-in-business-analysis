The problem

Sound familiar?

  • You keep saying "I should share what I'm learning" — but never actually start
  • Creating 3 content formats every week (post + slides + script) feels like too much work
  • AI-generated content doesn't sound like you — it's generic

I had all three problems at once. Then I built an agent for it.

02 / 08
The concept

What is a Claude Code Skill?

A reusable instruction file that Claude reads before generating anything — so it already knows your voice, your audience, and what you need.

Think of it like a briefing document. You write it once. Every time you run the skill, Claude is already briefed.

03 / 08
The skill structure

Three sections.
That's the whole skill.

Section 1

Persona

Who you are, your voice, your audience — loaded every run

Section 2

Before you start

Asks 4 questions first — topic, number, story, what to avoid

Section 3

Output

3 files saved to the right folders, named correctly, every time

No guessing. No generic output. It asks before it writes.

04 / 08
Running it

One command. Three files.

# Open terminal in your project folder
claude

# Type the skill name
❯ /weekly-content

# Claude asks 4 questions, then saves:
posts/01-your-topic/linkedin.md
posts/01-your-topic/slides.html
posts/01-your-topic/speaker-notes.md

06 / 08
Get started today

3 steps to set it up.

  1. Copy the agents/weekly-content/ folder into .claude/skills/ in your project
  2. Restart Claude Code — the skill is now available
  3. Type /weekly-content, answer 4 questions, review the 3 files
