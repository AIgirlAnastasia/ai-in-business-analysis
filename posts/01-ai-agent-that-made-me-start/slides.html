Finally Start Writing

How I built a personal AI agent with Claude Code
— and what it means for Business Analysts

AI girl Anastasia · March 2026

Sound familiar?

  • You spend hours preparing for meetings that could take 20 minutes
  • You review the same kind of document every week, manually
  • You know AI could help — but "that's for developers"

I said that last one for two years. Then I tried Claude Code.

What is Claude Code?

A terminal-based AI agent that reads your files, runs tasks, and saves results — all in plain English.

You describe what you need. It does it.

npm install -g @anthropic/claude-code
cd your-project
claude

No coding required. If you can write a clear email, you can use this.

Every prompt follows one framework

Take

What data do you give it?
Files, folders, a URL, text

Process

What should it do?
Find, classify, summarize, compare

Store

Where does the result go?
A file, a report, a folder

This is the same thinking BAs already do. The agent just executes it.

My real example: call analysis

In the calls/ folder are 11 meeting transcripts.
Read all of them. Find repeating problems.
For each one: what it is, who raised it, impact.
Save a structured report to report.md.

That's the entire prompt. The agent reads 11 files, finds patterns across all of them, and produces a structured report.

I turned this into a reusable Skill — one command that does this every time: /call-analysis

The actual difference

Before

Read 11 transcripts manually, take notes, look for patterns, format a report.

~3 hours

After

Type /call-analysis in the terminal. Review the result.

~8 minutes

The thinking is still mine. The reading and formatting is the agent's.

3 steps to try this today

  • Install Claude Code: npm install -g @anthropic/claude-code
  • Copy a skill from the GitHub repo into your project's .claude/skills/ folder
  • Run it: type /call-analysis (or any skill name) and press Enter
