The Module Pattern
Code counts. LLM interprets. You ask the questions.
1
GET
Connect your data source
Link Google Sheets via MCP. Claude reads the sheet directly — no copy-paste, no manual export.
Google Sheets MCP You: point to the sheet
2
WRITE
Ask Claude to write the script
Describe the question. Claude writes a formula into the sheet or a Python script — and inserts it directly.
Claude Agent You: ask the question
3
RUN
Execute — get exact numbers
Code runs, not the LLM. No hallucination possible. The formula calculates, the script outputs, the numbers are real.
Zero hallucination You: one command
4
EXPLAIN
Ask Claude what it means
Paste the output back. Claude spots the pattern, flags anomalies, suggests what to investigate next.
Claude Agent You: decide next step
